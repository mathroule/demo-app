# Demo App
Demo Android app with PR [Danger](https://danger.systems/ruby) and [SonarQube](https://sonarcloud.io) checks. 

# Executed checks
* PR classed as Work in Progress
* PR name (expected pattern `DEMO-123: PR label`)
* APK statistics
* Android lint
* Commit lint
* Checkstyle
* Findbugs
* JUnit tests
* Textlint
* Todoist
* SonarQube
